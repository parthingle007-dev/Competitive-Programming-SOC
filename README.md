# Competitive Programming Practice Repository

## Overview

This repository contains my solutions to competitive programming problems solved as part of my learning journey in Competitive Programming.

The repository is organized into three folders:

* **Sheet1** – Solutions to problems from Week 1 Sheet 1.
* **Sheet2** – Solutions to problems from Week 1 Sheet 2.
* **CSES** – Solutions to problems from the CSES Problem Set.

All solutions are implemented in **C++**.
